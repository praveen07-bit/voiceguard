# 🎙️ VoiceGuard — Audio Deepfake Detection System

An AI-powered web application that detects whether a voice recording 
is genuine or artificially generated using machine learning.

## Tech Stack
- Python, Flask, librosa, scikit-learn, SQLite

## How To Run
1. Install dependencies: pip install -r requirements.txt
2. Run: python main.py
3. Open: http://127.0.0.1:5000
